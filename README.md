## 功能特性

- **多角色配音** — 为每个角色单独配置参考音频，自动匹配朗读
- **场景配音** — 为每个场景单独配置场景音频，并智能循环播放
- **GAL 模式** — 仅朗读对话内容（格式：`[角色][表情][场景]:「对话」`）
- **听书模式** — 全文朗读，适合叙述性内容
- **行内播放按钮** — 在每条消息中注入逐句播放控件
- **悬浮播放器** — 进度条拖拽、倍速、音量控制，停靠右下角
- **自动推理 + 自动播放** — AI 回复后自动生成语音并播放
- **推理N句后自动播放** — 填写自定义句数后，推理完成自动播放
- **音频缓存** — IndexedDB 持久化，已推理内容无需重复请求
- **参考音频下拉选择** — 从后端 `ckyp/` 目录直接选择，无需手动上传
- **提示词注入** — 自动引导 AI 按指定格式输出，提升解析准确率
- **多预设管理** — 支持保存/切换多套配音方案
- **情感向量** — 支持8个维度的情感向量，生成更富有情感的角色语音

## 安装

1. 将本仓库克隆或下载到 SillyTavern 的扩展目录：
   ```
   https://github.com/xiaoxiongweihu/st-indextts2
   ```
   Index-tts-2.0 整合包下载：https://tcnlo9s668u9.feishu.cn/wiki/KRQ9wuqiViSOmJkfHxacNFfknjh
2. 启动 IndexTTS2 后端服务（默认端口 `7880`）
4. 在 SillyTavern 扩展面板中启用 **IndexTTS2 Player**

---

## 快速配置

1. 点击任意 AI 消息旁的 ⚙ 按钮，打开配音配置面板
2. 在角色列表中，从下拉框选择对应的参考音频文件
3. 开启「回复后自动推理」和「推理后自动播放」
4. 与 AI 对话，语音将自动生成并播放

> 参考音频文件放置于后端 `api/ckyp/` 目录，下拉框会自动列出所有可用文件。
> 场景音频文件放置于后端 `api/pjy/` 目录，每次重启酒馆都需要点击授权才能生效。

## 设置说明

| 设置项 | 说明 |
|--------|------|
| API 地址 | IndexTTS2 推理接口，默认： `http://127.0.0.1:7880/v1/audio/speech`；情感向量： `http://127.0.0.1:7880/api/v1/tts/tasks` |
| 音频列表地址 | 获取参考音频列表，默认 `http://127.0.0.1:7880/api/v1/voices` |
| 提示词注入 | 引导 AI 使用标准格式输出对话 |
| 解析模式 | `gal`（对话）或 `audiobook`（全文） |
| 回复后自动推理 | AI 回复完成后自动生成语音 |
| 推理完毕后自动续播 | 生成完成后自动开始播放（需页面有过用户交互） |
| 推理完N句后自动续播 | 填写自定义句数后，推理完成自动播放 |
| 默认音色 | 未配置角色时使用的参考音频文件名 |
| 默认速度 | 调节对话或全文语音的速度 |
| 全局音量 | 调节对话或全文语音的音量 |
| 参考音频&缓存管理 | 设置你参考音频文件所在文件夹 |
| 场景音效 | 设置你场景音效文件所在文件夹 |
| 场景音音量 | 调节场景音的音量 |
| 淡入淡出 | 调节场景音的淡入淡出的时间 |
| 同场景下循环播放场景音 | 开启后相同场景下循环播放 |
| 启用行内按钮 | 在消息文本中注入逐句播放控件 |

## 后端要求

- [IndexTTS2](https://github.com/index-tts/IndexTTS2) 推理服务，运行在本地
- 服务需提供以下接口：
  - `POST /v1/audio/speech` — 文字转语音
  - `GET /api/v1/voices` — 获取参考音频列表
  - `POST /api/v1/upload` — 上传参考音频
